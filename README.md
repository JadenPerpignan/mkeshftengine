## MKESHFT ENGINE

### Built by Jaden Perpignan

**MKESHFT ENGINE** is a modern, scriptable 2D game engine written in C++23 + SDL, designed for fast iteration and real shipping workflows.  
It is the spiritual successor to [Wintermoon](https://github.com/wintermoon/wintermoon), reimagined for today’s desktop, mobile, and web targets.

### Why MKESHFT? ⚡

**MKESHFT** evokes *makeshift*: practical tools, creative momentum, and building with purpose.  
The engine keeps your pipeline lean so you can focus on game feel, gameplay loops, and shipping.

### Features 🎮

- Lua scripting for rapid gameplay iteration
- Cross-platform runtime (Linux, Windows, macOS, WebAssembly, Android, iOS)
- Data-driven project layout with scenes, objects, assets, particles, and fonts
- Production-friendly stack powered by proven open-source libraries

### Platform Support 🌍

- Desktop: Linux, macOS, Windows
- Web: WebAssembly in modern browsers
- Mobile: Android and iOS

### Under the Hood 🛠️

MKESHFT ENGINE builds on battle-tested libraries including boost, Box2D, EnTT, stb, yyjson, sol2, PhysFS, SDL, and miniaudio.

### Quick Start 🚀

1. Install dependencies and tools
2. Build the engine
3. Run your cartridge
